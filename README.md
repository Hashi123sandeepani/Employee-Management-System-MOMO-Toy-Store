# Employee Management System (MOMO Toy Store)

A console-based Java application built to help a toy store manage employee records, with separate Admin and Employee roles.

## Tech Stack

- **Language:** Java
- **IDE:** NetBeans
- **Data Handling:** Java Collections

## Key Features

- **Role-Based Access** — distinct Admin and Employee functionality
- **Auto-Generated Employee IDs** — unique ID assigned to each new employee
- **Full CRUD Operations** — Add, Update, Delete, and Search employee records
- **Salary-Based Sorting** — employees sorted by salary in ascending order
- **Input Validation** — handles empty fields, negative salaries, and non-numeric inputs

## How to Run

1. Clone this repository:
2. Open the project folder in **NetBeans IDE**.
3. Build and run the project (`Run` → `Run Project`, or press F6).

## Project Structure

- `src/model/` — data models (Employee)
- `src/service/` — business logic (SystemManager)
- `src/momtoystoremanagemetsystem/` — application entry point (Main)
